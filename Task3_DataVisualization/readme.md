# Task 3 - Data Visualization

## Overview
Created 6 beautiful visualizations on 1000 books 
dataset from books.toscrape.com

## Tools Used
- Python
- Matplotlib
- Seaborn

## Charts Created
- Chart 1: Rating Distribution (Bar Chart)
- Chart 2: Price Distribution (Histogram)
- Chart 3: Average Price per Rating (Bar Chart)
- Chart 4: Top 10 Most Expensive Books (Horizontal Bar)
- Chart 5: Price vs Rating (Boxplot)
- Chart 6: Rating Distribution (Pie Chart)

## Key Insights
- Most Common Rating: 3 Stars
- Average Price: £35.07
- Most Expensive Book: £59.99
- Cheapest Book: £10.00
